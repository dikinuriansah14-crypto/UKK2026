# Sistem Pengaduan Sekolah Frontend

Website frontend untuk sistem pengaduan sekolah dengan fitur lengkap.

## Struktur Folder
- `index.html` - Halaman Utama
- `submission.html` - Formulir Laporan
- `tracking.html` - Lacak & Riwayat
- `faq.html` - Pertanyaan Umum
- `css/style.css` - Styling umum
- `js/` - JavaScript files
- `assets/` - Gambar dan file statis

## Fitur
- Responsive Design (Bootstrap)
- Dark/Light Mode
- Form Wizard dengan validasi
- Drag-and-Drop Upload
- Real-time Stats Counter
- Timeline Status
- Chat-like Interaction
- Searchable FAQ
- Skeleton Loading
- FAB untuk WhatsApp

## Cara Menjalankan
Buka `index.html` di browser web modern.

Untuk development, gunakan live server extension di VS Code.

## Catatan
- reCAPTCHA perlu API key untuk implementasi penuh
- Backend integration diperlukan untuk fungsi penuh
- Enkripsi client-side menggunakan placeholder
